## 3. 推奨の進め方（次の2〜3手）

1. **修正S＋追加Sのセット**: F1(攻速を実効化) + N1(ポーションクイックスロット) + V1(アイコン暫定差し替え) — 小粒3つで1コミット圏
2. **N2 スキル個別効果** — 中規模。属性/状態異常/ボス戦と全部噛み合う本命
3. その後に **N3 エンドゲーム**（大物）か **N5 固有絵**（協働フロー）を並走
