## 3. クエスト70本の構成（各ランク10本）

内訳テンプレート（ランクごとに適用）:
1. **看板狩猟 3〜5本** — ホームボス・亜種の単体狩猟（ボス数に応じて）
2. **出張狩猟 2本** — 下位ランクのボスが別アリーナに強化出現（歴戦フラグ or レベル倍率）
3. **連続狩猟 2本** — 2体を順に狩る（例: R3「炎と狼」= ヴルム→フェンリル）
4. **露払い 1本** — 雑魚の群れ殲滅→ボス出現（objectives を順次処理）
5. **歴戦 1本** — そのランクの看板の歴戦個体。報酬・レア率アップ

命名規則: `hunt_r{ランク}_{連番}_{slug}`（例: `hunt_r3_07_flame_then_wolf`）。
全70本 repeatable。既存12本はIDを維持したまま新体系のランクへ整理統合する。

## 4. 素材（23種 → 59種）

ファミリーごとに3段構成（モンハン式）:
- **通常素材** = 既存のボス素材を流用（竜鱗・業火の核 など）
- **上質素材**（新規12種、R5–R7）= 討伐で確定1〜2個
- **レア素材**（新規12種、R8–R9、逆鱗ポジション）= 低確率（5〜8%、歴戦で2倍）

亜種素材（新規12種）= 亜種ごとに1種（色違い・属性違いの素材）。

例（竜ファミリー）: 竜鱗（既存R7）→ 竜の剛翼（上質R7）→ 竜王の逆鱗（レアR9）＋ 黒竜の暗鱗（亜種R8）。

新規素材合計: 12＋12＋12 = 36種 → 総計59種。名称・レアリティは実装時に `items.json` へ、ドロップは `drops.json` の各ボステーブルへ。

## 5. 装備調整（両方やる）

### 5a. 既存高ティアの材料再配線
月影〜覇王など Lv40 以上の汎用ティアのレシピに、対応レベル帯のボス素材を1〜2種混ぜる（例: 炎帝シリーズ ← 業火の核/ヴルム上質素材、冥王シリーズ ← モルド素材）。「高い装備は大型狩猟で作る」導線にする。

### 5b. 固有効果シリーズ（新規 約42点 + レシピ）
ファミリーごとに 武器1〜2種 + 防具2〜3部位。同Lv帯の汎用装備より一段強いが特化型。素材はそのボス（＋亜種）の狩猟でのみ入手。

|---|---|
| 主（樹）| maxHp・def（堅牢） |
| 番人（岩）| def・magDef |
| ノクス／クルオル | 吸血（新ステータス）・critRate |
| スライム／オーラム | maxMp・金運（新ステータス） |
| ヴルム／アズヴルム | physAtk・critRate（火力特化） |
| フェンリル／スコル | atkSpeed・moveSpeed（速攻） |
| ヴァルガ／ヴァルガノス | 全ステータス微強化（万能） |
| スポラ／ミアズマ | dropRate・maxHp |
| ガロ／イグニガロ | def・physAtk（重戦士） |
| グラキエス／フレアリス | magAtk・maxMp |
| モルド／ルクス | magDef・maxHp／accuracy |
| 深淵 | 最終盤の全能装備 |

| ヒュドラ | accuracy・atkSpeed |

**新ステータス2種**（Phase 0 で `computeDerived` に追加）:
- `lifesteal` 吸血: 与ダメージの n% を HP 回復（戦闘側フック1箇所）
- `goldRate` 金運: ゴールド獲得 +n%（獲得処理フック1箇所）

## 6. 技術対応（Phase 0）

1. **連続スポーン**: hunt クエストの objectives（kill 複数）をアリーナで順次消化。1体目撃破→演出→2体目スポーン。露払い（雑魚n体→ボス）も同じ仕組みで処理
2. **歴戦倍率**: クエスト定義 `veteran` → スポーン時に HP/攻撃/報酬倍率を適用
3. **クエストボードのランクフィルタ**: 修正済みのチップバー部品を流用（R1〜R7 チップ）
4. **新ステータス**: lifesteal / goldRate（stats.ts 集約 + フック2箇所 + 装備差分UI表示）
5. **validate-data 拡張**: 亜種の基本種参照・素材3段の存在・クエストIDの命名・ランクとminLevelの整合をチェック
6. **テスト**: 連続狩猟の進行ロジック、歴戦倍率、素材ドロップ、新ステータスの計算（すべて Phaser 非依存で）

## 7. フェーズ分割

- **Phase 0 骨組み**: 上記6項目。既存12クエストの回帰確認
- **Phase 1 R1〜R3**: ボス9体（亜種5＋新規1＋既存整理）、素材、装備シリーズ（該当ファミリー）、クエスト30本
- **Phase 2 R4〜R7**: 残りボス、クエスト40本、高レベル帯の材料再配線とバランスパス
